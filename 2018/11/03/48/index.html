2018-11-03 正睿 阅读量

Description

开始有一个空的集合S，进行Q次操作：

1.插入一个数x（如果已存在就忽略）
2.删除一个数x（如果不存在就忽略）
3.询问点集中数与x的最短距离
定义两个数a,b的距离为，每次将a乘一个质数或除以一个质数因子，使a变成b的最小步数。$（Q≤50000,x≤1000000，时限2秒）$
